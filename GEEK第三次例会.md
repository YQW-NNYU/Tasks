# <mark>Win回收站机制</mark>

1、第一个8 bytes是$I文件的头部
2、第二个8 bytes保存了被删除文件的尺寸
3、第三个8 bytes保存了删除文件的时间
4、后面的520 bytes内容是文件被删除之前的完整路径以及原文件名。



删除文件时，系统会在当前用户当前分区的回收站中创建一个文件$IXXXXXX.EXT，同时将删除的文件重命名为$RXXXXXX.EXT并移动至回收站
